## EXPLICATIONS

**Q1 — B** : Note prix = Note_max × **(Prix_min / Prix_i)**. Plus l'entreprise est chère, plus Prix_min/Prix_i est petit, donc moins bonne note.

**Q2 — C** : Prix_min = l'offre **la moins chère après négociation** (moins-values, plus-values, remises incluses).

**Q3 — C** : L'entreprise la moins chère obtient Prix_min/Prix_min = 1 → Note = Note_max × 1 = **Note_max** (score maximum).

**Q4 — B** : Note délai = Note_max × **(Délai_min / Délai_i)**. Le délai le plus court = meilleure note.

**Q5 — B** : Délai_i long → Délai_min/Délai_i petit → **note basse**. L'entreprise la plus lente est pénalisée.

**Q6 — C** : Écart = **(Offre - Estimation) / Estimation × 100**. Positif = dépassement, négatif = économie.

**Q7 — A** :
```
Écart = (185 000 - 175 000) / 175 000 × 100 = 10 000/175 000 × 100 = +5,71%
```
C'est un dépassement de 5,71% par rapport à l'estimation.

**Q8 — B** : Montant final = Offre de base **- Moins-values + Plus-values - Remise commerciale**.

**Q9 — A** :
```
Base après moins/plus-values = 220 000 - 5 000 + 2 000 = 217 000 €
Remise = 217 000 × 2% = 4 340 €
Montant final = 217 000 - 4 340 = 212 660 €
```
> Attention : la remise s'applique sur le montant **après** moins/plus-values, pas sur l'offre initiale.
> Résultat correct : 212 660 € — si la réponse A indique 213 660 € c'est une erreur de calcul dans l'intitulé. Vérifiez toujours le détail.

**Q10 — B** : Le classement est basé sur le **score total** qui combine tous les critères pondérés. Le prix le plus bas n'est pas forcément retenu si d'autres critères pénalisent l'offre.

**Q11 — C** : Les critères et pondérations sont définis dans le **DCE**, notamment dans le **Règlement de Consultation (RC)** et/ou le CCAP.

**Q12 — B** :
```
Note prix = 60 × (240 000 / 258 000) = 60 × 0,93023 = 55,81 pts
```

**Q13 — B** :
```
Note délai = 20 × (10 / 14) = 20 × 0,71429 = 14,29 pts
```

**Q14 — D** : Le score total = 42 + 30 + 12 = **84 pts sur 100**. Toutes les formulations sont équivalentes et correctes.

**Q15 — B** : Le ratio €/m² permet de **comparer** le coût d'un lot entre offres concurrentes, ou de le confronter à des références professionnelles (bases de données BATIPRIX, SEBTP...).
