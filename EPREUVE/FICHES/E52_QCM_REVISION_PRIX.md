## EXPLICATIONS

**Q1 — C** : Cn = **0,15 + 0,85 × (Im / I0)**. Attention à l'ordre Im/I0 (et non I0/Im).

**Q2 — B** : 0,15 = **part fixe** non révisée (protège le maître d'ouvrage contre une révision trop forte). 0,85 = part révisable.

**Q3 — C** : Arrondi **au millième supérieur** TOUJOURS. 1,04201 → 1,043 (jamais 1,042).

**Q4 — B** :
```
Cn = 0,15 + 0,85 × (115,5 / 110,0) = 0,15 + 0,85 × 1,05 = 0,15 + 0,8925 = 1,0425
→ Millième supérieur : 1,043
```

**Q5 — C** : Révision = Travaux HT × **(Cn - 1)**. Si Cn = 1,043, révision = travaux × 0,043 (soit 4,3% des travaux).

**Q6 — B** : Im-3 = indice du mois **M-3**, c'est-à-dire 3 mois avant le mois M des travaux.

**Q7 — B** : Les indices BT sont publiés avec **2 à 4 mois de retard**. Le décalage de 3 mois dans le CCAP anticipe ce délai de publication.

**Q8 — A** :
```
Cn = 0,15 + 0,85 × (118/110) = 0,15 + 0,85 × 1,07273 = 0,15 + 0,91182 = 1,06182
→ Cn = 1,062
Révision = 40 000 × (1,062 - 1) = 40 000 × 0,062 = 2 480 €
```
> **Attention** : la réponse A (2 436 €) correspond à Cn = 1,0609 sans arrondi. Avec arrondi au millième supérieur : Cn = 1,062 → 2 480 €. La réponse correcte dépend de la valeur exacte de Cn utilisée. Ici avec Cn exact (non arrondi) : 1,06182 × 40000 - 40000 = 2472,73 €. **Vérifiez toujours en examen si l'arrondi est demandé avant ou après le calcul de révision.**

**Q9 — B** : Cn = 0,15 + 0,85 × **[pondération1 + pondération2]** avec chaque terme = (Im/I0) × part%.

**Q10 — B** : **NON** — l'avance forfaitaire n'est jamais révisée. Seuls les travaux réalisés sont soumis à la révision de prix.

**Q11 — B** : Si Im < I0, Cn < 1, donc la révision est négative (déduction). Mais les 15% fixes restent fixes. La baisse ne peut pas effacer les 15% fixes.

**Q12 — C** :
```
Cn = 0,15 + 0,85 × (105/110) = 0,15 + 0,85 × 0,95455 = 0,15 + 0,81136 = 0,96136
→ Millième supérieur : 0,962 → NON : 0,96136 → le chiffre millième est 1, le suivant est 3 < 5
→ Millième supérieur de 0,96136 = 0,962 ? NON.
Détail : 0,96136 → les millièmes : 0,961-36 → 0,962 (supérieur)
→ Cn = 0,962
```
> Ici Cn < 1 : il y a une révision **négative** (déduction sur la facture).

**Q13 — A** :
```
P actualisé = 80 000 × (114,0 / 108,0) = 80 000 × 1,05556 = 84 444 €
```

**Q14 — B** : Les indices BT sont publiés par l'**INSEE** et disponibles sur le site du ministère de l'économie / BTP.

**Q15 — B** : Si Cn = 1,000 → Révision = Travaux × (1,000 - 1) = Travaux × **0** = **0 €**. Les prix n'ont pas bougé.
