## PARTIE 1 — QUESTIONS SUR LE CORPUS

### Etape 1 : Lire et annoter (25 min)

Pour CHAQUE document, noter sur le brouillon :
```
Doc 1 : [Auteur] — [Titre] — [Date] — [Nature : article/essai/temoignage/image]
  → Idee principale : ..................
  → Idee secondaire : ..................

Doc 2 : [Auteur] — [Titre] — [Date] — [Nature]
  → Idee principale : ..................
  → Idee secondaire : ..................

(idem pour docs 3 et 4)
```

### Etape 2 : Repondre aux questions

**TYPES DE QUESTIONS FREQUENTES :**

1. **Question de comprehension** (1 document)
   → Gabarit : "Dans ce document, l'auteur [nom] met en evidence [idee]. Il souligne que [detail]. Cette analyse permet de comprendre que [interpretation]."

2. **Question de confrontation** (2+ documents)
   → Gabarit : "Les documents [X] et [Y] abordent tous deux la question de [theme]. Cependant, alors que [auteur X] considere que [idee X], [auteur Y] estime au contraire que [idee Y]. Cette divergence s'explique par [raison]."

3. **Question sur une image**
   → Gabarit : "Cette image represente [description factuelle]. Elle illustre [idee] en montrant [detail visuel]. Ce document vient confirmer / nuancer / contredire la these de [auteur] dans le document [X], selon laquelle [idee]."

### Phrases TYPES a memoriser (reutilisables dans TOUTES les copies)

```
PRESENTER UN DOCUMENT :
  "Le document [X], [nature] de [auteur] publie en [date], traite de [theme]."
  "Dans cet extrait, [auteur] developpe l'idee selon laquelle [these]."

CONFRONTER DEUX DOCUMENTS :
  "A l'instar de [auteur 1], [auteur 2] souligne que [point commun]."
  "En revanche, [auteur 2] se distingue de [auteur 1] en affirmant que [difference]."
  "Si [auteur 1] adopte une vision [optimiste/critique], [auteur 2] propose une approche plus [nuancee/radicale]."

CITER UN TEXTE :
  "Comme le souligne [auteur], « [citation courte] » (l. [numero])."
  "Selon [auteur], « [citation] » (doc. [X], l. [numero])."
  → TOUJOURS : guillemets + numero de ligne
  → JAMAIS plus de 2 lignes de citation

CONCLURE UNE REPONSE :
  "Ainsi, les documents du corpus permettent de comprendre que [bilan]."
  "En definitive, cette confrontation met en lumiere [synthese]."
```

## PARTIE 2 — ECRITURE PERSONNELLE

### Structure OBLIGATOIRE (a suivre mecaniquement)

```
INTRODUCTION (5-8 lignes)
  1. Phrase d'accroche (fait d'actualite, citation, question)
  2. Reformuler la question posee
  3. Annoncer le plan : "Nous verrons d'abord que..., puis que..., enfin que..."

PARTIE I — OUI / THESE (10-15 lignes)
  → 1 argument + 1 exemple concret
  → 1 argument + 1 exemple du corpus

PARTIE II — MAIS / ANTITHESE (10-15 lignes)
  → 1 argument + 1 exemple concret
  → 1 argument + 1 exemple du corpus

PARTIE III — DONC / DEPASSEMENT (8-10 lignes)
  → Nuancer : la verite est entre les deux
  → 1 argument + 1 exemple personnel ou culturel

CONCLUSION (3-5 lignes)
  1. Bilan : repondre a la question posee
  2. Ouverture : poser une nouvelle question liee
```

### Gabarit complet (a recopier et adapter le jour J)

```
INTRODUCTION :
"[Fait d'actualite ou constat general sur le theme]. Cette question est au
coeur du corpus propose, qui interroge [reformuler la question]. Nous
examinerons d'abord [axe 1], avant de nuancer en montrant que [axe 2],
pour enfin proposer [axe 3]."

TRANSITION 1→2 :
"Neanmoins, cette vision merite d'etre nuancee. En effet, [transition]."

TRANSITION 2→3 :
"Des lors, il convient de depasser cette opposition en considerant que [transition]."

CONCLUSION :
"En definitive, [reponse a la question]. Toutefois, on peut se demander
si [ouverture sous forme de question]."
```

## ORTHOGRAPHE — 20 REGLES A CONNAITRE

```
1.  a / à       → "a" = verbe avoir (il a) | "à" = preposition (à Paris)
                   Test : remplacer par "avait" → si ca marche = "a"

2.  et / est    → "et" = addition | "est" = verbe etre
                   Test : remplacer par "etait" → si ca marche = "est"

3.  er / é      → "er" = infinitif | "é" = participe passe
                   Test : remplacer par "vendre/vendu" → "vendre" = er, "vendu" = é

4.  ce / se     → "ce" = demonstratif (ce livre) | "se" = pronom reflechi (il se lave)

5.  ces / ses   → "ces" = demonstratif pluriel | "ses" = possessif (ses affaires)

6.  ou / où     → "ou" = choix (rouge ou bleu) | "où" = lieu/temps (la ville où)

7.  la / là     → "la" = article/pronom | "là" = lieu (je suis là)

8.  leur / leurs → "leur" devant verbe = invariable | "leurs" devant nom = pluriel

9.  Accord sujet-verbe : TOUJOURS chercher le sujet
    "Les problemes que pose cette question SONT nombreux" (pas "est")

10. Accord participe passe avec ETRE : accorder avec le sujet
    "Les solutions SONT PROPOSEES" (pas "proposé")

11. Accord participe passe avec AVOIR : accorder avec COD AVANT
    "Les idees que j'AI DEVELOPP-EES" (COD "idees" avant = feminin pluriel)
    "J'AI DEVELOPPE des idees" (COD apres = pas d'accord)

12. Pluriel en -s : la plupart des noms et adjectifs
13. Pluriel en -x : bijoux, cailloux, choux, genoux, hiboux, joujoux, poux
14. Pluriel en -aux : journaux, travaux, chevaux (sauf bals, festivals, carnavals)

15. Tout / tous / toute / toutes
    "TOUS les documents" (masculin pluriel)
    "TOUTES les idees" (feminin pluriel)

16. Quel / quelle / quels / quelles : accorder avec le nom
    "QUELLE que soit la solution" (feminin)

17. Meme : variable quand adjectif, invariable quand adverbe
    "Les MEMES idees" / "Ils sont MEME venus" (adverbe = invariable)

18. Ni... ni... → verbe au PLURIEL
    "Ni l'auteur ni le journaliste ne SONT d'accord"

19. C'est / s'est : "c'est" = cela est | "s'est" = il s'est (reflechi)

20. Ponctuation : TOUJOURS un espace AVANT et APRES : ; ! ?
    (en francais seulement, pas en anglais)
```

## CONNECTEURS LOGIQUES (a apprendre par coeur)

```
ADDITION :      de plus, en outre, par ailleurs, egalement, qui plus est
OPPOSITION :    cependant, neanmoins, toutefois, en revanche, or
CAUSE :         en effet, car, puisque, etant donne que, du fait de
CONSEQUENCE :   par consequent, ainsi, des lors, c'est pourquoi, de ce fait
CONCESSION :    certes... mais, bien que, malgre, quoique, il est vrai que
ILLUSTRATION :  par exemple, notamment, ainsi, tel que, c'est le cas de
CONCLUSION :    en definitive, en somme, pour conclure, finalement, en resume
TEMPS :         tout d'abord, ensuite, puis, enfin, dans un premier temps
```
