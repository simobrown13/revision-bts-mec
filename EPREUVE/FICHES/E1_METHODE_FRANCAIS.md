## PARTIE 1 — QUESTIONS SUR LE CORPUS (10 pts)

### Etape 1 : Lire et annoter (15 min)

Pour CHAQUE document, noter au brouillon :
```
Doc 1 : [Auteur] — [Titre] — [Date] — [Nature : article/essai/roman/image]
  → Idee principale : ..................
  → Position de l'auteur : pour / contre / nuance

Doc 2 : idem
Doc 3 : idem
(Doc 4 si present)
```

### Etape 2 : Definir les mots cles de la question

```
⚠️ LE CORRECTEUR VALORISE cette demarche !
Avant de repondre, ecrire : "Le mot [X] peut se definir comme [definition]"
Exemple : "Le mot 'progres' designe ici aussi bien l'avancee technique
que l'amelioration des conditions de vie."
```

### Etape 3 : Repondre aux questions

**QUESTION DE COMPREHENSION (1 document) :**
```
Gabarit :
"Dans ce document, [auteur] [nature du texte : defend/analyse/temoigne de]
[idee principale]. Il souligne que [detail important] en precisant que
[element secondaire]. Cette approche permet de comprendre que [interpretation]."
```

**QUESTION DE CONFRONTATION (2+ documents) — RAPPORTE LE PLUS DE POINTS :**
```
⚠️ CONFRONTER = montrer points communs ET differences. JAMAIS resumer chaque doc separement.

Gabarit CONFRONTATION :
"Les documents [X] et [Y] abordent tous deux la question de [theme commun].
[Auteur X] considere que [idee X], comme en temoigne [citation courte « ... » (l. X)].
De son cote, [auteur Y] [verbe : rejoint/nuance/contredit] cette vision en
affirmant que [idee Y] (doc. Y, l. X).
Ainsi, si les deux auteurs s'accordent sur [point commun], ils divergent
quant a [point de divergence]."

Variantes de formulations :
  ACCORD    : "A l'instar de [X], [Y] souligne que..."
  NUANCE    : "[Y] rejoint partiellement [X] mais nuance en..."
  OPPOSITION: "En revanche, [Y] s'oppose a [X] en affirmant que..."
  COMPLEMENT: "[Y] enrichit la reflexion de [X] en ajoutant que..."
```

**QUESTION SUR UNE IMAGE :**
```
Gabarit :
"Ce document iconographique represente [description factuelle : qui, quoi, ou].
[Element visuel remarquable] evoque [interpretation].
Cette image [confirme/nuance/illustre] la these de [auteur] dans le document [X],
selon laquelle [idee]."
```

### Reflexes anti-erreurs Partie 1

```
❌ ERREUR 1 : Recopier des phrases entieres du texte
✅ CORRECT  : Reformuler dans vos propres mots + citation courte entre « »

❌ ERREUR 2 : Resumer chaque document separement sans les confronter
✅ CORRECT  : Toujours relier les documents entre eux (accord/opposition/nuance)

❌ ERREUR 3 : Donner son avis personnel dans la Partie 1
✅ CORRECT  : Rester OBJECTIF et NEUTRE dans les questions sur le corpus

❌ ERREUR 4 : Ne pas citer les numeros de ligne
✅ CORRECT  : Toujours indiquer (l. 12) ou (doc. 2, l. 5)
```

## PARTIE 2 — ESSAI (10 pts)

### Ce que dit la grille officielle

```
LE CORRECTEUR CHERCHE :
  1. Vous repondez AU SUJET (pas a cote)
  2. Votre reflexion est NUANCEE (pas un seul point de vue)
  3. Votre texte PROGRESSE de facon coherente (pas d'idees en vrac)
  4. Vous utilisez le CORPUS + des references CULTURELLES + des exemples PERSO
  5. Votre langue est correcte

LE CORRECTEUR NE DEMANDE PAS :
  → Pas de plan impose (these/antithese est un OUTIL, pas une obligation)
  → Pas de nombre de parties impose
  → "Sans contraintes formelles fixes" (citation officielle)
```

### Structure RECOMMANDEE (pas obligatoire mais efficace)

```
INTRODUCTION (5-8 lignes)
  1. Phrase d'accroche (fait, constat, question)
  2. Reformuler le sujet / definir les mots cles
  3. Annoncer la progression : "Nous examinerons d'abord..., avant de..."

DEVELOPPEMENT — 2 ou 3 mouvements (pas de nombre impose)
  Mouvement 1 : Premier aspect de la reflexion
    → 1 argument + 1 exemple (corpus OU culture OU perso)
    → Transition vers le mouvement suivant

  Mouvement 2 : Deuxieme aspect (nuance, approfondissement)
    → 1 argument + 1 exemple different du precedent
    → Transition

  Mouvement 3 (optionnel) : Depassement ou ouverture
    → 1 argument + 1 exemple

CONCLUSION (3-5 lignes)
  1. Bilan : repondre a la question posee
  2. Ouverture : poser une question plus large
```

### Gabarits prets a remplir

**INTRODUCTION :**
```
"[Constat general ou fait d'actualite sur le theme]. Cette question, au coeur
du corpus propose, invite a s'interroger sur [reformuler le sujet].
Nous examinerons d'abord [axe 1], avant de montrer que [axe 2]."
```

**TRANSITIONS :**
```
Mouvement 1 → 2 : "Neanmoins, cette vision merite d'etre nuancee."
                   "Cependant, on peut aussi considerer que..."
                   "Toutefois, cette approche presente des limites."

Mouvement 2 → 3 : "Des lors, il convient de depasser cette opposition."
                   "En definitive, ne faudrait-il pas plutot envisager..."
```

**CONCLUSION :**
```
"En definitive, [repondre a la question en 1-2 phrases].
Toutefois, on peut se demander si [ouverture — nouvelle question liee]."
```

## ORTHOGRAPHE — 20 REGLES ESSENTIELLES

```
1.  a / à       "a" = avoir (il a) | "à" = preposition (à Paris)
                 Test : remplacer par "avait" → si OK = "a"

2.  et / est    "et" = addition | "est" = etre
                 Test : remplacer par "etait" → si OK = "est"

3.  er / é      "er" = infinitif | "é" = participe passe
                 Test : remplacer par "vendre/vendu" → "vendre" = er, "vendu" = é

4.  ce / se     "ce" = demonstratif (ce livre) | "se" = reflechi (il se lave)

5.  ces / ses   "ces" = demonstratif pluriel | "ses" = possessif

6.  ou / où     "ou" = choix | "où" = lieu/temps

7.  la / là     "la" = article/pronom | "là" = lieu

8.  leur / leurs "leur" devant verbe = invariable | "leurs" devant nom = pluriel

9.  Accord sujet-verbe : TOUJOURS chercher le sujet
    "Les problemes que pose cette question SONT nombreux"

10. Participe passe avec ETRE : accorder avec le sujet
    "Les solutions SONT PROPOSEES"

11. Participe passe avec AVOIR : accorder avec COD AVANT
    "Les idees que j'AI DEVELOPPEES" (COD avant = accord)
    "J'AI DEVELOPPE des idees" (COD apres = pas d'accord)

12. Pluriel en -s : la plupart des noms et adjectifs
13. Pluriel en -x : bijoux, cailloux, choux, genoux, hiboux, joujoux, poux
14. Pluriel en -aux : journaux, travaux, chevaux

15. Tout/tous/toute/toutes : accorder avec le nom
16. Quel/quelle/quels/quelles : accorder avec le nom

17. Ni... ni... → verbe au PLURIEL
18. C'est / s'est : "c'est" = cela est | "s'est" = reflechi

19. TOUJOURS un espace AVANT et APRES les signes doubles : ; ! ?
20. Majuscule apres un POINT, jamais apres une virgule
```

## CONNECTEURS LOGIQUES (a apprendre par coeur)

```
ADDITION :      de plus, en outre, par ailleurs, egalement
OPPOSITION :    cependant, neanmoins, toutefois, en revanche, or
CAUSE :         en effet, car, puisque, etant donne que
CONSEQUENCE :   par consequent, ainsi, des lors, c'est pourquoi
CONCESSION :    certes... mais, bien que, malgre
ILLUSTRATION :  par exemple, notamment, ainsi, tel que
CONCLUSION :    en definitive, en somme, pour conclure, finalement
TEMPS :         tout d'abord, ensuite, puis, enfin
```
