- Im BT34 = **147,9**

**Étape 2 : Calcul du coefficient Cn provisoire**

> Cn = 0,15 + 0,85 × [(140,5 / 120,8) × 10% + (147,9 / 122,9) × 90%]

Détail :
- 140,5 / 120,8 = **1,16308**
- 147,9 / 122,9 = **1,20342**
- (1,16308 × 0,10) + (1,20342 × 0,90) = 0,11631 + 1,08308 = **1,19939**
- 0,85 × 1,19939 = **1,01948**
- 0,15 + 1,01948 = **1,16948**

Arrondi au millième supérieur : **Cn provisoire = 1,170**

**Étape 3 : Montant de la révision situation 6**

> Montant HT révisé = 13 693,59 × 1,170 = **16 021,50 €**

## POINTS DE VIGILANCE RÉCURRENTS (erreurs classiques)

- **Note pondérée VT** : (note/10) × poids en % du total. Ex : 8/10 avec poids 20% → (8/10) × 20 = 16 pts
- **Offre moins-disante** : c'est le prix le plus BAS après application des formules → ici 127 750 €
- **Formule prix** : note = 10 × (1 − écart/moins-disante) → donne 10/10 au moins cher
- **Millième SUPÉRIEUR** pour Cn : 1,16948 → **1,170** (pas 1,169)
- **Indices provisoires** : si l'indice du mois d'exécution n'est pas publié, utiliser le **dernier indice connu**
- **Avenant = ST × 1,15** : ne pas confondre le montant du sous-traitant (5 463,74) avec l'avenant (6 283,30)
- **Paiement direct** : le sous-traitant est payé par le MOA → le coût est quand même comptabilisé dans le CD de l'entreprise
- **Compte-prorata oublié** : erreur classique de chiffrage → à intégrer systématiquement dans les FOp
- **RB%** : calculé sur le montant HT **réel** (52 479,64) pas le montant de base (44 070,92)
